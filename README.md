### 🎞 소개
---

* Hi Kitty: 블록체인 기반 길고양이 기부 플랫폼
* 길고양이 공존: 길고양이에 대한 사람들의 일상생활의 피해를 최소화함과 인식개선 효과
* 기부 활성화: 기부 신뢰성 향상으로 인한 기부활성화 효과


  </br>

### ✔ 시작하기

---

```shell
# /backend/
$ gradlew clean build
$ build/libs/hi-kiity-0.0.1-SNAPSHOT.jar -jar app.jar

# /frontend/
$ npm install
$ npm start
```
